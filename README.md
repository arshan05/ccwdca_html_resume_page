**Single Page Resume Website (HTML Only)**

**Overview**

This project is a single-page resume website built using only HTML.
No external CSS or JavaScript is used, as per the assignment instructions.

The resume is structured clearly using proper HTML tags like headings, sections, lists, and tables to make it easy to read and understand.

All the content (name, skills, experience, projects, etc.) is sample data and can be customized.

**Features**

The resume includes the following sections:

Header with name and role

About Me section

Skills list (with nested lists)

Experience table

Projects section

Education table

Contact details

Footer

The layout focuses on:

Clean structure

Proper spacing

Readability

Correct use of HTML semantics

**Technologies Used**

HTML5 only

(No CSS framework, no JavaScript, no libraries)

**Project Structure**
```bash
/ccwdca_html_resume_page
│
├── index.html
└── README.md
```

**How to Run the Project**

Download or clone the project

Open the index.html file

Double-click it or open it using any web browser (Chrome, Edge, Firefox, etc.)

That’s it — no setup required.

**Usage**

This page can be used as a basic resume website

You can edit the text inside index.html to add your own details

The structure can be reused for other resumes or HTML practice
